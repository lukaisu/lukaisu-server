Lukaisu Server

Connect to your server

Connecting to your own Lukaisu Server? It has to allow this app. Add this to the server’s environment and restart it: 

CORS_ALLOWED_ORIGINS=https://localhost

Also check the address is reachable from this device — on a home network use the server’s IP, e.g. http://192.168.1.20:8010.
